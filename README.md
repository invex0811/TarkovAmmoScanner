# Tarkov Ammo Scanner

Windows-приложение-компаньон для Escape from Tarkov. Оно делает снимок области возле курсора, читает название патрона через OCR, сопоставляет его с базой Tarkov.dev и показывает компактную карточку поверх игры.

## Что уже работает

- `Ctrl + Shift + A` — сканирование названия патрона возле курсора.
- `Ctrl + Shift + D` — демонстрационная карточка без запуска игры.
- Компактный оверлей в стиле внутриигрового интерфейса.
- Урон, пробитие, урон броне и оценка эффективности против 1–6 классов брони.
- Загрузка актуальных данных из JSON API Tarkov.dev.
- Русская и английская локализация названий.
- Локальный кэш, чтобы приложение запускалось после первой успешной загрузки даже без API.
- Сохранение последнего снимка OCR для отладки.

## Требования

- Windows 10/11.
- Python 3.11 или новее.
- Tesseract OCR с языками `rus` и `eng`.
- Escape from Tarkov в оконном или безрамочном режиме.

## Быстрый запуск

Откройте PowerShell в папке проекта:

```powershell
Set-ExecutionPolicy -Scope Process Bypass
.\scripts\setup.ps1
.\run.ps1
```

Скрипт создаст `.venv`, установит Python-зависимости и попробует установить Tesseract через `winget`, если он не найден.

## Как проверить

1. Запустите приложение командой `.\run.ps1`.
2. Нажмите `Ctrl + Shift + D` — должна появиться тестовая карточка.
3. Запустите Tarkov в Borderless или Windowed.
4. Откройте окно осмотра патрона.
5. Наведите курсор на значок лупы слева от названия.
6. Нажмите `Ctrl + Shift + A`.

Приложение захватывает прямоугольник справа от курсора, поэтому курсор должен находиться около лупы или начала названия.

## Если патрон не распознаётся

Последний снимок сохраняется сюда:

```text
%LOCALAPPDATA%\TarkovAmmoScanner\debug\last_scan.png
```

Проверьте:

- выбран ли русский язык OCR;
- установлен ли файл `rus.traineddata`;
- находится ли курсор возле лупы;
- не перекрыто ли название другим окном;
- используется ли оконный или безрамочный режим игры.

## Техническая схема

```text
Горячая клавиша
  → снимок области возле курсора
  → увеличение и контрастирование
  → Tesseract OCR
  → нечёткое сопоставление с патронами
  → компактный topmost overlay
```

Приложение не читает память игры и не взаимодействует с игровым процессом. Оно работает только со снимком экрана.

## Ограничения первой версии

- Положение области OCR пока фиксировано относительно курсора.
- Эффективность против классов брони — компактная эвристическая оценка 0–5, а не симуляция конкретной брони и её текущей прочности.
- Для первого запуска нужна сеть, чтобы загрузить полный список патронов.
